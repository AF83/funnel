[What's funnel aims to be](https://github.com/AF83/funnel/blob/master/doc/Architecture.md).

## Installing things

First, Elixir must be installed, check out the resources section.

``` shell
make install
```

## Running things

``` shell
iex -S mix server
```

## Testing things

``` shell
mix test
```
